AssetVerse

AssetVerse is a full-stack asset management system where organizations can manage company assets and employees can request, track, and manage assigned assets. The application includes role-based dashboards (HR & Employee), authentication, asset lifecycle management, and a Stripe-powered subscription upgrade system.
